# IT Club Management System

## Introduction

**IT Club Management System** is a C-based program to manage IT club members and events efficiently. It offers a user-friendly interface for administrators to handle member registration, event management, and communication within the club.

---

## Features

* Secure **user login**
* **Add, delete, search, edit** members
* **Add and delete events**
* Display members and events (sorted by ID)
* Menu-driven interface with **error handling**

---

## Tools & Technologies

* **Language:** C
* **IDE:** Code::Blocks

---

## Installation

1. Open the project in Code::Blocks.
2. Compile and run the program.
